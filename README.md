# Super Store Sales Dashboard

## Overview

The Super Store Sales Dashboard is a Power BI project created to analyze sales performance, profit trends, customer behavior, and regional business performance. The dashboard provides clear business insights using interactive charts, KPIs, and visual reports.

---

## Project Objectives

* Analyze overall sales and profit performance
* Track key business KPIs
* Understand customer segments and purchasing trends
* Identify top-performing product categories
* Monitor shipping and payment preferences
* Support business decisions using data insights

---

## Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling
* Data Visualization

---

## Dashboard Features

* Total Sales Analysis
* Profit Analysis
* Order Quantity Tracking
* Category and Sub-category Performance
* Customer Segment Analysis
* State-wise Profit Distribution
* Payment Mode Insights
* Shipping Mode Analysis
* Sales Trend Analysis
